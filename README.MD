# Password Generator

This is a simple web-based application that generates random passwords. It allows users to select the length of the password and then generates a password accordingly. Users can also copy the generated password to their clipboard with the click of a button.

## Features

- Input field for displaying the generated password.
- Buttons for generating a password and copying it to the clipboard.
- Slider for selecting the length of the password.
- Utilizes Font Awesome for icons.
- Responsive design.

## Usage

1. Open `index.html` in your web browser.
2. Adjust the password length using the slider.
3. Click on the "Create Password" button to generate a password.
4. Click on the "Copy" button to copy the generated password to your clipboard.
